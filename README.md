# videojs-chapter-nav

video.js plugin for skip to next/previous chapter buttons.

Built for video.js 5.x and above.

## Usage

```scss
@import "~videojs-chapter-nav/src/plugin";
```

```js
require('videojs-chapter-nav')
videojs(element, {
  plugins: {
    chapterNav: {}
  }
})
```
